# Ne İzlesem - Movie & Series Recommendation Platform

A full-stack web application that provides mood-based movie and TV series recommendations. Users can create accounts, discover content through curated lists, and filter productions based on their current mood.

**Live URL:** [https://ne-izlesem.com/](https://ne-izlesem.com/)

## Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript (Responsive Design)
* **Backend:** Node.js, Express.js
* **Database:** MongoDB (NoSQL)
* **Deployment:** Live production server with Google Search Console indexing.

## Key Features
* **User Authentication:** Secure sign-up and login system for personalized experience.
* **Mood-Based Filtering:** Unique filtering system allowing users to find content based on their current mood (e.g., "Moduna Göre Seç").
* **Curated Collections:** Specialized categories such as "Must Watch" (Kesin İzle) and "Time Killer" (Can Sıkıntısında Akar).
* **Random Picker:** Feature to suggest a random movie for undecided users.
* **Responsive Interface:** Fully optimized for desktop and mobile devices.

## Roadmap (Future Improvements)
* **User Interaction:** Implementing a comment and rating system for movies.
* **User Profiles:** Advanced profile management.
* **Code Refactoring:** Migrating the backend structure to a strict MVC architecture.
